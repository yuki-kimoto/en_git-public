git init - Gitリポジトリの初期化

Gitリポジトリを初期化するには「git init」コマンドを使用します。 

git init

perlzemiというディレクトリをGitリポジトリとして初期化するサンプルです。cdコマンドで「perlzemi」ディレクトリに移動してから「git init」コマンドを実行します。 

cd perlzemi
git init

初期化すると「.git」というディレクトリが作成されます。 

.git

「.git」ディレクトリの中には、このリポジトリに関するGitの設定や、Gitのコミット情報が保存されます。

ワーキングディレクトリではない生のGitリポジトリを作成する

ワーキングディレクトリを持たない生のGitリポジトリを作成するには「git init」の「--bare」オプションを使用します。

開発者として使う場合は、生のGitリポジトリ(.gitに該当するディレクトリだけ)を作成する必要はないと思いますが、Githubのようなリポジトリ管理Webインターフェースを開発する場合には、作業ディレクトリを持たないGitリポジトリを作成する場合があります。 

mkdir foo.git
cd foo.git
git init --bare
