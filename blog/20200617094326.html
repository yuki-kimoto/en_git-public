GithubのWikiの編集

GithubのWikiを編集してみましょう。右上の「Edit」をクリックします。

Wikiのコンテンツの編集方法や、保存方法は、GithubのWikiの新規作成で解説したものと同じです。

Wikiの書き方 - マークダウン記法の基礎

Wikiを書く場合にマークダウン記法を使いますが、書き方をいくつか覚えておきましょう。

見出し

見出しは以下です。 

# 見出し1
## 見出し2
### 見出し3

コード

コード。 

```
use strict;
use warnings;
```

リスト

リスト。 

* 項目1
* 項目2
* 項目3

テーブル

テーブル。 

|  TH  |  TH  |
| ---- | ---- |
|  TD  |  TD  |
|  TD  |  TD  |

リンク

ページリンク。Wikiに存在しないページ名の場合は、リンクが赤くなり、ページの新規作成ページにリンクされます。 

[[ページ名]]

ページ名と異なるタイトルをつける場合。 

[[環境構築手順書|20200618102400]]

外部リンク。 

[[木本システム|https://kimoto-system.co.jp/]]

Gitは、分散リポジトリ型のソースコードの管理ツールです。ブランチの作成速度は高速です。
ソースコードの変更履歴のすべてをリモートリポジトリだけでなく、ローカルのリポジトリも保存しています。このため簡易的なファイルのバックアップ用途としても使えます。
Gitは、Githubなどのクラウド型ソースコード管理システムの基盤です。
opensslなど他のUnix/Linuxミドルウェアと同じく、Gitは主にC言語で記述されており、Perlがサポートツールとして、一部利用されています。
