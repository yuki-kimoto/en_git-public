Github Wikiのコンテンツのバックアップ方法

Github WikiはGitで管理されているので、WikiのGitリポジトリを「git clone」するだけです。ページ全体をバックアップできます。 

git clone https://github.com/yuki-kimoto/perlzemiubuntu.wiki.git

ローカルで編集してpushする

GithubのWikiは単なるGitリポジトリなので、ローカルで編集して「git push」するということも可能です。

全体的に修正したい内容がある場合に、便利です。

Gitは、分散リポジトリ型のソースコードの管理ツールです。ブランチの作成速度は高速です。
ソースコードの変更履歴のすべてをリモートリポジトリだけでなく、ローカルのリポジトリも保存しています。このため簡易的なファイルのバックアップ用途としても使えます。
Gitは、Githubなどのクラウド型ソースコード管理システムの基盤です。
opensslなど他のUnix/Linuxミドルウェアと同じく、Gitは主にC言語で記述されており、Perlがサポートツールとして、一部利用されています。
