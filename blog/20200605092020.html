git init - Gitリポジトリの初期化

Gitリポジトリの初期化を行うには「git init」を使用します。Gitリポジトリにしたいディレクトリの中で「git init」を実行します。 

cd myapp
git init

cdコマンドで、myappというディレクトリの中に移動して、「git init」を実行しています。

「.git」というGitがリポジトリ管理に利用する隠しディレクトリが作成されます。lsコマンドで確認できます。 

ls -a

「.git」という隠しディレクトリができています。
