GithubでのGitリポジトリの作成

次に、Gitリポジトリを作成します。Githubにサインインしていない場合は、サインインしてください。

Gitリポジトリの作成画面へ移動

Gitリポジトリの作成画面への移動は、右上の「+」の「New repository」から行います。

Gitリポジトリの作成

プロジェクト名を入力して「Create repository」のボタンを押します。

成功すると以下の画面が表示されます。

これで、Github上でのGitリポジトリは完成です。

公開鍵の登録

次に必要な手順は、公開鍵を登録する作業です。GithubへのリポジトリをSSHプロトコル上で、反映させるためには、公開鍵認証を行う必要があります。

HTTPSでも、リポジトリをpush/pullを行うこともできますが、pushのたびにパスワードを聞かれるので、開発を行う場合は、公開鍵を登録して、SSLプロトコルを使うのがお勧めです。

ローカルの既存のGithubレポジトリをGithub上のリポジトリに反映する

ローカルの既存のGithubレポジトリをGithub上のリポジトリに反映するには、リポジトリの作成成功画面に記載されている、以下のコマンドを実行します。 

git remote add origin git@github.com:yuki-kimoto/testproject.git
git push -u origin master
