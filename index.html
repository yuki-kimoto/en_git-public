実践Git入門

Git講座 - プロのITエンジニアのソースコード管理

実践Git入門は、プロのWeb開発者の現場で使われているソースコード管理ツールGitを学べる講座です。GitはLinuxサーバー上でWebアプリケーションのプログラムのソースコードやHTML・CSSを管理するために実務で広く活用されているコマンドラインツールです。 

# ソースコードをコミット(ファイル管理情報に追加)する
git commit -m "Fix web api bug"

# ソースコード管理情報をリモートサーバーに転送
git push origin main

# チームの他のメンバーが修正した内容を取り込む
git merge tanaka_web_api_fix

Git関連情報

Gitは、分散リポジトリ型のソースコード管理システムです。
リモートとローカルで完全に同一のソースコード管理情報を保有できるため、バックアップが簡単。
ブランチの作成が高速。完全な履歴を保有しているので、ソースコードをいつでも過去の状態に戻すことができます。

PerlによるWebシステム開発、データ分析、ディープラーニングを使ったAIのソースコード管理に活用できます。GitはLinuxミドルウェアのひとつで、主にC言語で書かれ、部分的にPerlで書かれています。
