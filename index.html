Perl開発者のGit入門

Perl開発者のGit入門は、Perl開発者が効率よくソースコード管理ツールGitを学べる講座です。GitはLinuxサーバー上でWebアプリケーションのプログラムのソースコードやHTML・CSSを管理するために実務で広く活用されているコマンドラインツールです。 

# ソースコードをコミット(ファイル管理情報に追加)する
git commit -m "Fix web api bug"

# ソースコード管理情報をリモートサーバーに転送
git push origin main

# チームの他のメンバーが修正した内容を取り込む
git merge tanaka_web_api_fix
